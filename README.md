# Vue 3 + TypeScript + Vite


## 建構指令

### Install
````
yarn install
````
### Run
````
yarn run dev
````
### Build
````
yarn build
````

### Test
````
yarn test
````

## 專案 Import Alias
````
'@': 'src'
'@pages':'src/pages'
'@assets':'src/assets',
'@comps': 'src/components',
'@utils': 'src/utils',
'@router': 'src/router',
'@store': 'src/store',
'@css': 'src/css',
'@tools': 'src/tools',
````
